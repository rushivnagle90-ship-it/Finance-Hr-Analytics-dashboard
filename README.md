# Finance-Hr-Analytics-dashboard
An enterprise-grade HR Dashboard designed to reduce attrition and optimize hiring. Includes Department-wise Performance tracking, City wise, Salary Band Distribution, and Manager-level RLS.

Key Features
HR Analytics: AI-based Attrition Risk scoring (High/Medium/Low) aur Department-wise performance tracking.
Finance P&L: Full Profit & Loss statement jisme YoY (Year-over-Year) variance aur Revenue vs Budget analysis hai.
Executive View: Senior leadership ke liye 'Revenue per Employee' aur 'HR Cost Ratio' jaise high-level KPIs.
Row-Level Security (RLS): Dynamic security implement ki gayi hai taaki Managers sirf apne department ka data dekh sakein.

Tech Stack
Tool: Power BI Desktop
Data Modeling: Star Schema (Fact and Dimension tables)
DAX: Time Intelligence, Dynamic Ranking, aur Conditional Formatting.
Preview: Interactive HTML dashboard for quick web viewing.
